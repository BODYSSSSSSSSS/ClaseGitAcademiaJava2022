Git vs Gitlab

GIT es un sistema de control de versiones. GitHub, GitLab, BitBucket, etc. Son las plataformas en las cuales podemos alojar el código fuente de nuestros proyectos.

Git es como una paqueteria

El funcionamiento de git podemos relacionarlo con el funcionamiento de una agencia de envios como DHL, Fedex o Estafeta.

Ya que en estas se tiene un inventario / bitacora de los productos a enviar.

Git hace algo similar, solo que git en lugar de enviar productos o cartas, este envia archivos, registos y cambios
